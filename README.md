> "Where creativity builds the future."

A modern, high-performance personal portfolio and digital services platform for **Chima**, a Software Developer & AI Solutions Architect specializing in turning manual chaos into automated clarity.

## 🚀 Overview

Chima.dev is a showcase of intelligent systems and human-centered design. Built with a "dark mode first" aesthetic, it leverages modern web technologies to deliver a fast, responsive, and future-ready user experience.

### Core Services:
- **Web Design & Redesign**: Sleek, responsive sites focused on conversion and performance.
- **App Development**: Custom web applications designed to solve real business problems.
- **AI Automation & Chatbots**: Smart systems for customer engagement and lead generation.
- **Workflow Optimization**: Streamlining internal processes to boost productivity.

---

## 🛠️ Technology Stack

The project prioritizes performance and maintainability through a clean, modern stack:

- **Core**: Semantic HTML5
- **Logic**: Vanilla JavaScript (ES6+)
- **Styling**: Tailwind CSS & Vanilla CSS (via `brand.css` overrides)
- **Typography**: **JetBrains Mono** — A monospaced font for a high-performance, developer-centric feel.
- **Tools**: Firebase, Antigravity, Claude AI, Gemini CLI, Next.JS, Vercel, Stripe.

## 💻 Local Development

To preview the project locally, you can use any static file server.

### Using `npx http-server`:
```bash
npx http-server -p 8080 .
```

### Using Python:
```bash
python -m http.server 8080
```

Once the server is running, navigate to `http://127.0.0.1:8080` in your browser.
